# 📖 Subssum <a name="about-project"></a>

Subssum is a financial services platform or app focusing on transactions and payments.It includes features like transaction history, card management, and possibly airtime and data purchases for mobile networks.The project uses React for frontend development, with a focus on responsive design for both desktop and mobile interfaces.It incorporates modern UI elements such as gradients, icons, and a clean, user-friendly layout.The platform handle various payment methods including transfers, wallet payments, and card payments, while tracking transaction statuses and details.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

A working connection and a coding editor like visual studio code

### Setup

Clone this repository to your desired folder:

- git@github.com:Megagig/subssum.git

Use git push at your desired location and use the command git clone and paste the URL

### Install

Install this project with:

- cd driver-license-portal
- npm install

### Usage

To run the project, execute the following command:

- npm run start

### Run tests

To run tests, run the following command:

- cd src
- npm run test
